## 網頁修改思路

老師給的基礎網頁功能已經很完善，不過我覺得作為使用者看到這個網頁可能沒辦法很快知道有哪些功能，所以在標題下增加了一行說明包含舉例。

在Query Builder的區域也增加了範例按鈕，讓使用者還不知道怎麼操作或要輸入什麼的時候，有更直接的例子可以點選，更快速的知道這區塊的功能。

LoTUS-BF這個名字雖然很特別，會吸引人的好奇心，但我個人偏好能更直接知道這個網頁是在做什麼的名稱，所以更改成了NeuroBridge，代表這是一個可以連接有關神經的資訊的系統。


最後，我增加了一個彈跳視窗，在進入網站的最一開始就能讓使用者直接了解網頁的內容和功能。
